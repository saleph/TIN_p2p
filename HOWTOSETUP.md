## Preparation

### Boost
1. Download [Boost C++ lib](http://www.boost.org/doc/libs/1_65_1/more/getting_started/unix-variants.html).
2. Unzip boost_1_65_1 at home dir `~/`.
3. `$ ./bootstrap.sh`
4. `$ ./b2`

### Python2
```$ sudo apt install python```

Python has to be ver 2.7!

### Scons
```$ sudo apt install scons```

## Compilation

```
$ scons [{tests, monitor}]
```
